# Battle Stage Progression System Plan

## Context
This plan documents a comprehensive redesign of the battle UI and core combat mechanics for 乘法小鎮 (Multiplication Town), an educational multiplication game. The goal is to implement a 9-stage progression system where each stage introduces specific multiplication table complexity.

## 4. Gameplay Loop

### 4.1 Battle Flow
```
START BATTLE
    │
    ▼
LOAD STAGE (apply difficulty modifiers)
    │
    ▼
GENERATE QUESTION (based on stage table)
    │
    ▼
START TIMER (countdown with visual feedback)
    │
    ├─► CORRECT ANSWER ─► Deal damage, add combo, spawn particles
    │                         │
    │                         ├─► MONSTER DEFEATED? ─► VICTORY → Rewards
    │                         │
    │                         └─► Next question
    │
    ├─► WRONG ANSWER ─► Take damage, reset combo
    │                         │
    │                         └─► HERO DEFEATED? ─► DEFEAT → Retry
    │
    └─► TIME UP ─► Wrong answer handling
```

### 4.2 Question Generation by Stage
| Stage | Table | Question Types | Mix Ratio |
|-------|-------|---------------|-----------|
| 1 | 2x | Direct only | 100% direct |
| 2 | 3x | Direct only | 100% direct |
| 3 | 4x | Direct, missing factor | 80% / 20% |
| 4-5 | 5-6x | Direct, missing factor | 70% / 30% |
| 6-7 | 7-8x | Direct, missing, application | 60% / 25% / 15% |
| 8-9 | 9-10x | All types, harder numbers | 50% / 30% / 20% |

### 4.3 Damage Calculation
```
Base Damage = Hero ATK (50-80 based on level)
Speed Bonus = 1.0 + (timeRemaining / totalTime) * 0.5
Combo Multiplier = 1.0 + (combo * 0.1), max 2.0
Stage Modifier = 1.0 (base)

Final Damage = Base × Speed Bonus × Combo Multiplier × Stage Modifier
```

### 4.4 Enemy HP Scaling
```
Base HP = 50
Stage HP = Base HP × (1 + (stage - 1) × 0.15)
Boss HP = Stage HP × 2.5
```

## 7. Stage-Specific Features

### 7.1 Stage 1 (Tutorial - Integrated)
- Larger timer (4.5s base instead of 3s)
- No combo penalty (combo displays but doesn't reset on wrong)
- Celebratory particles on correct answers
- Tutorial popups explaining UI elements
- "訓練模式" banner
- Must achieve ≥50% accuracy to complete (not strict failure)
- After Stage 1 clear, combo system activates fully

### 7.2 Stages 2-5 (Learning Phase)
- Standard timer scaling
- Introduce combo system
- Progressive HP increase
- Unlock achievements per stage

### 7.3 Stages 6-8 (Challenge Phase)
- Reduced timer (0.9x)
- Higher HP scaling
- Introduce "Speed Bonus" visual feedback
- Boss encounter hints

### 7.4 Stage 9 (Final Boss)
- 0.85x timer
- 2.5x HP multiplier
- Multi-phase battle (2 monsters)
- Dramatic entrance animation
- Unique boss sprite
- Special victory celebration
