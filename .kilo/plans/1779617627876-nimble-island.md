### Step 1: Create `updateArcaneBar()` function
Location: After `triggerMagicBurst()` (around line 5615)

```javascript
function updateArcaneBar() {
  const arcaneBar = document.getElementById('arcaneBar');
  if (!arcaneBar) return;

  // Calculate fill percentage based on combo (max at 20)
  // Combo 5 = 25%, Combo 10 = 50%, Combo 15 = 75%, Combo 20 = 100%
  const maxCombo = 20;
  const fillPct = Math.min((battleState.combo / maxCombo) * 100, 100);
  arcaneBar.style.width = fillPct + '%';
}
```
