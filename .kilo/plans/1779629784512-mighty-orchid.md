## 2. Combat System Mechanics

### 2.1 Core Loop

```
┌─────────────────────────────────────────────────────────┐
│                    BATTLE TURN STRUCTURE                │
├─────────────────────────────────────────────────────────┤
│  1. PLAYER PHASE (Answer Math Question)                │
│     ├─ Question displayed (2-3 sec timer)               │
│     ├─ Player input answer                              │
│     └─ Math accuracy → Damage calculation               │
│                                                         │
│  2. DAMAGE PHASE                                        │
│     ├─ Base damage = f(hero ATK range)                  │
│     ├─ Speed bonus = f(responseTime)                    │
│     ├─ Combo multiplier = f(comboCount)                 │
│     └─ Hero passive effects                             │
│                                                         │
│  3. MONSTER RESPONSE PHASE                              │
│     ├─ Monster HP reduced                               │
│     ├─ If HP <= 0: Monster defeated                     │
│     └─ If HP > 0: Monster attacks (if timer expires)   │
│                                                         │
│  4. REPEAT until Monster HP = 0 OR Hero HP = 0          │
└─────────────────────────────────────────────────────────┘
```

### 2.2 Turn Structure

| Turn Component | Specification |
|----------------|--------------|
| **Timer Duration** | Base: 3 seconds, modified by stage.timerMod (0.85x-1.5x) |
| **Answer Input** | Text input or multiple choice (configurable per stage) |
| **Correct Answer** | Deal damage, increment combo, proceed |
| **Wrong Answer** | No damage dealt, combo reset, monster may attack |
| **Timeout** | Treated as wrong answer, combo reset |

### 2.3 Damage Calculation

```javascript
finalDamage = baseDamage * speedMultiplier * comboMultiplier * passiveBonus

Where:
- baseDamage = random(hero.atkMin, hero.atkMax) + (heroLevel - 1) * 3
- speedMultiplier = clamp(1.0 + (timeRemaining / timerDuration) * 0.5, 1.0, 1.5)
- comboMultiplier = 1.0 + floor(comboCount / 5) * 0.2  // +0.2 per 5 combo
- passiveBonus = hero-specific modifier
```

### 2.4 Math Accuracy Impact

| Response Time | Speed Bonus | Educational Rationale |
|---------------|-------------|---------------------|
| < 1 second | 1.5x damage | Reward instant recall (fluency) |
| 1-2 seconds | 1.25x damage | Reward quick recall |
| 2-3 seconds | 1.0x damage | Standard recall |
| > 3 seconds | 0.5x damage | Encourages mastery |

**Design Principle**: Faster = better reward, but accuracy always takes priority. A correct answer at any speed deals damage; speed only multiplies the reward. This prevents frustration while still incentivizing fluency practice.

### 2.5 Monster Turn Mechanics

| Condition | Monster Action | Hero Damage |
|-----------|---------------|------------|
| Wrong answer | Basic attack | 10-15 HP |
| Timeout | Basic attack | 10-15 HP |
| Combo ≥ 10 | Skips turn | 0 (player advantage) |
| Shield available | Attack blocked | 0 |

### 3.2 Hero-Specific Skill Trees

#### Knight (🗡️ 銀藍騎士)

**Theme**: Defensive tank with shield mechanics

| Skill | Level | Cost | Cooldown | Effect |
|-------|-------|------|----------|--------|
| Shield Block | 1 (default) | 0 | 2 turns | Reduce next monster damage by 50% |
| Challenge | 3 | 10 mana | 3 turns | Monster always attacks Knight for 3 turns |
| Bulwark | 7 | 20 mana | 4 turns | Gain shield equal to 20% max HP |
| Last Stand | 12 | 0 mana | 5 turns | When HP < 25%, gain 50% ATK boost |
| Divine Shield | 18 | 50 mana | 8 turns | Invulnerable for 1 turn |

**Passive - Block**: 15% chance to completely block monster attack

#### Mage (🔮 銀月法師)

**Theme**: High damage glass cannon with mana system

| Skill | Level | Cost | Cooldown | Effect |
|-------|-------|------|----------|--------|
| Arcane Bolt | 1 (default) | 5 mana | 1 turn | Deal 150% normal damage |
| Mana Surge | 4 | 0 mana | 3 turns | Restore 50% max mana |
| Arcane Barrage | 8 | 25 mana | 3 turns | Deal 3 hits at 50% damage each |
| Spell Shield | 11 | 15 mana | 4 turns | Block magic attacks (future enemy types) |
| Arcane Mastery | 15 | 40 mana | 5 turns | Next answer deals 300% damage |
| Mana Font | 20 | 0 mana | 6 turns | Passive: Gain 2 mana per correct answer |

**Passive - Crit Boost**: 15% chance to deal 200% damage (base 10%)

#### Berserker (⚔️ 烈焰狂戰)

**Theme**: High risk/high reward with rage mechanics

| Skill | Level | Cost | Cooldown | Effect |
|-------|-------|------|----------|--------|
| Wild Swing | 1 (default) | 0 | 1 turn | Deal 120% damage, but take 5 HP self-damage |
| Rage Build | 5 | 0 | 2 turns | Gain "Rage" stack (max 5) |
| Berserker's Roar | 9 | 15 mana | 4 turns | Consume all Rage: each stack = +25% damage |
| Bloodlust | 13 | 20 mana | 4 turns | Heal for 30% of damage dealt this turn |
| Undying Rage | 17 | 30 mana | 6 turns | When HP would be 0, survive with 1 HP (once per battle) |

**Passive - Rage**: Starting at 3 combo, gain +5% damage per combo. Resets on taking damage.

### 3.3 Mana System

| Parameter | Value |
|-----------|-------|
| Max Mana | 100 |
| Mana per correct answer | 15 |
| Mana per wrong answer | -10 |
| Mana overflow | Does not carry over |

**Mana Regeneration**: At the start of each player turn, recover 10 mana.

## 5. Educational Integration

### 5.1 Math Learning Alignment

| Game Element | Educational Purpose | Implementation |
|--------------|-------------------|----------------|
| Question generation | Practice multiplication facts | Direct (a×b=?), missing factor (a×?=c), application (story problems) |
| Timer pressure | Build fluency through timed practice | 3-second base, adjustable per difficulty |
| Combo system | Encourage sustained focus and practice | Track consecutive correct answers |
| Speed bonus | Motivate instant recall over counting | Damage multiplier based on response time |
| Error feedback | Identify learning gaps | Wrong answers highlight weak tables |
| Progress tracking | Show improvement over time | Per-table accuracy statistics |

### 5.2 Adaptive Difficulty

| Performance | Adjustment |
|-------------|------------|
| 5 consecutive correct | Increase question difficulty (higher tables or missing factor) |
| 3 consecutive wrong on same table | Show visual hint, offer easier questions |
| Accuracy < 60% over 10 questions | Suggest practice mode, reduce timer pressure |
| Accuracy > 90% over 20 questions | Enable harder question types, faster timer |

### 5.3 Accessibility Features

| Feature | Purpose |
|---------|---------|
| Input mode toggle | Allow choice between typing or multiple-choice |
| Option count (2-4) | Adjust difficulty by changing number of choices |
| Timer adjustment | Modify time pressure for different learners |
| Visual multiplication arrays | Show dot patterns to reinforce conceptual understanding |
| Cantonese voice feedback | Audio reinforcement for Cantonese-speaking learners |
