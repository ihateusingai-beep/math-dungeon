## ⚔️ 階段二：核心戰鬥系統

### 2.1 攻擊回合（手速輸入）

```
題目：7 × 8 = ?
時限：3秒緩衝
倍率：輸入越快 → 傷害越高（最高 1.5x）
```

**題目類型變化：**
- 基礎題：`6 × 7 = ?`
- 缺數題：`6 × ? = 42`
- 應用題：`有3群怪物，每群8隻，共幾隻？`

### 2.2 防禦回合（精準判斷）

- 不是單純選答案
- **多選一 + 干擾項**：「判斷這題對不對」（訓練驗算）
- **護盾機制**：答對多題累積護盾，抵擋怪物大招

### 2.3 Combo 系統

| 連續答對 | 效果 |
|----------|------|
| 3題 | 🔥 火焰拳（小範圍） |
| 5題 | ✨ 魔法大爆發（範圍伤害） |
| 10題 | 🌟 終極魔法 |

**觸發反饋：**
- 答對：`叮！` + 數字飛出動畫
- Combo：`哇！` + 爆炸特效
- 答錯：輕微 `噗`（避免挫折）

## 📈 階段三：成長與進度系統

### 3.1 貨幣與經驗

| 貨幣 | 來源 | 用途 |
|------|------|------|
| 🪙 金幣 | 打怪、每日任務 | 升級HP/攻擊/速度 |
| ⭐ 經驗值 | 完成關卡 | 解鎖新技能 |

### 3.2 角色進化樹

```
Lv.1 學徒法師 → Lv.10 初級魔法師 → Lv.20 元素大師
外觀改變 + 新技能（逐步解鎖除法練習）
```

### 3.3 成就系統

- 完美通關某關
- 連擊 20 次
- 收集所有怪物卡片
- 解鎖新皮膚或故事章節

## 🗺️ 階段四：關卡與探索設計

### 4.1 地圖結構

```
🌲 魔法森林（5關） → ⛰️ 雲端山谷（5關） → 🏰 暗黑城堡（5關） → 👹 魔王廳（1關）
難度：1-5表 → 6-9表 → 綜合混合
```

### 4.2 非線性支線

- 幫 NPC 完成任務：`"村莊有 4 群羊，每群 7 隻..."`
- 獎勵：額外金幣 + 知識卡片

### 4.3 Boss 戰設計

```
階段1：抵禦怪物攻擊（答題）
階段2：累積護盾
階段3：發動Combo反擊
```

## 🎓 階段五：教育深度與適應性

### 5.1 自適應難度

| 狀態 | 調整 |
|------|------|
| 連錯 2 次 | 給提示或簡化題目 |
| 正確率 > 90% | 加快題目速度 |
| 正確率 < 50% | 降低難度 |

### 5.2 錯題本

- 戰鬥後顯示「弱點表格」
- 可以重點練習特定乘法
- 家長/老師可查看進度

### 5.3 家長/老師模式

| 功能 | 用途 |
|------|------|
| 後台報表 | 看哪個乘法最弱 |
| 遊玩時間 | 监控學習時長 |
| 正確率追蹤 | 了解學習進度 |
| 多玩家合作 | 兩人一起打Boss |

## 🏆 階段六：留存與動機

### 6.1 每日任務

| 任務 | 獎勵 |
|------|------|
| 完成 10 題 | 🪙 50 金幣 |
| 累計登入 7 天 | 🔥 連續登入徽章 |
| 打敗 3 個怪物 | ⭐ 100 經驗 |

### 6.2 收藏元素

- **怪物圖鑑**：打敗後解鎖怪物卡
- **魔法書**：每解鎖一頁故事
- **證書系統**：生成「乘法大師證書」可列印分享

### 6.3 防止流失設計

| 問題 | 解法 |
|------|------|
| 答錯太挫折 | 扣血但給重試機會 |
| 關卡太長 | 每關 3-5 分鐘 |
| 重複性高 | 題目隨機化 + 特效變化 |
