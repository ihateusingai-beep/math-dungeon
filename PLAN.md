# Math Dungeon MVP 計劃書
> 版本：v1.0 | 日期：2026-05-25
> 目標：模組化數學 RPG，MVP 只做乘法範疇

---

## 🎯 MVP 範圍

### ✅ MVP 包含
- 首頁 + 用戶輸入名字 + 角色選擇
- 五大範疇入口（其他範疇顯示「敬請期待」）
- **乘法範疇** → 子項 → 3 種難度 → 關卡
- 乘法的完整題庫（2-9 乘法表 + 進階）
- 回合制 RPG 戰鬥（進攻/防守）
- 初階：3 選 1 | 中階：4 選 1 | 高階：Numpad 輸入
- 等級、經驗、金幣、簡單商店
- Admin Page（關卡開關）
- 多人模式架構（預留介面，單人先行）

### ❌ MVP 排除（預留架構）
- 其他四大範疇（代數/度量/圖形/數據處理）— 框架預留，內容留空
- 多人即時對戰 — 預留房間/匹配介面
- 技能樹複雜系統 — 精簡技能卡
- 高級動畫 — 基本 CSS 動畫即可

## ⚔️ 戰鬥系統

### 回合制流程
```
我方回合：
  ├─ [普通攻擊] → 答對 → 扣怪物血
  ├─ [使用技能] → 消耗 MP → 放大招
  └─ [防守] → 下回合攻擊力+50%

敵方回合：
  └─ 自動攻擊 → 扣我方血

答題環節：
  ├─ 答對：造成傷害 + (暴擊率加成)
  └─ 答錯：怪物反擊
```

### 數值系統
| 參數 | 數值 |
|------|------|
| 初始 HP | 100 |
| 初始 MP | 50 |
| 初始 ATK | 10 |
| 等級經驗公式 | 等級² × 100 |
| 金幣獲得 | 10 + 等級 × 2 |
| 答對獎勵 | XP + 硬幣 |

### 技能（精簡版）
| 技能 | MP | 效果 |
|------|-----|------|
| 火球術 | 15 | 固定 30 傷害 |
| 治療術 | 20 | 回 30 HP |
| 暴擊之刃 | 10 | 2倍傷害 |
| 聖光護盾 | 15 | 防守 2 回合 |

## 📊 Admin Page

### 功能
1. **關卡開關** — 個別乘法關卡 enable/disable
2. **題目管理** — 新增/編輯/刪除題目
3. **怪物設定** — 修改 HP/ATK/金幣獎勵
4. **全局設定** — 經驗倍率、是否顯示提示

### 資料結構（stages.json）
```json
{
  "multiplication": {
    "2": {
      "name": "2的乘法",
      "stages": [
        { "id": "2-1", "name": "2的乘法·初", "enabled": true, "monsters": ["slime"], "questions": [...] },
        { "id": "2-2", "name": "2的乘法·中", "enabled": true, "monsters": ["wolf"], "questions": [...] },
        { "id": "2-3", "name": "2的乘法·終", "enabled": true, "monsters": ["ogre"], "questions": [...] }
      ]
    },
    "3": { ... },
    ...
    "addition": { "enabled": false, "stages": [] },
    ...
  }
}
```

### Admin 登入
- 簡單密碼保護（`admin123`，可改）
- 暫存 localStorage，未來可換 Firebase
